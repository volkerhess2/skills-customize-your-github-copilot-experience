# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build the classic word-guessing game using Python strings, loops, and user input. Players will guess letters to reveal a hidden word before running out of attempts, practicing key programming concepts.

## 📝 Tasks

### 🛠️ Build a Hangman Game

#### Description

Create a Hangman game where players guess letters to reveal a hidden word before running out of attempts. Your program should randomly select words from a predefined list and provide interactive gameplay with progress tracking.

#### Requirements

Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses and display current progress in underscore format (_ _ _)
- Track incorrect guesses remaining and display them
- End game when word is guessed or attempts are exhausted
- Display clear win/lose messages
- Use string manipulation, loops, and conditionals throughout the implementation
